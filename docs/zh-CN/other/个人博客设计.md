# 个人博客的制作技术

需要前后端都用到；

# 前端需要的技术

做一个个人博客前端需要基本的html,css ,javascript这是最基本的语法要会，然后想把界面做的美观且快速，可以用bootstarp框架，做后台需要很多组件交互的功能，这个时候就需要用到react或者vue了，我选择了vue，vue相关框架提供很多的组件可以使用，我用的是element-plus，vue和bootstrap的区别是啥呢，一类是把html,css,js整个打包组件化，一个是只关注界面效果，不在乎能不能打包，所以vue虽然功能强大，但是官网并没有组件库；

还有一个layui它也是一个动态框架，类似vue但是它的观念和vue ,react是不同的，他强调原生用法，而不提供额外的语法，vue和react都是受mvvc，视图和数据绑定显示的观念影响创建的；layui的使用上就像jquey一样，但是它的功目的却和jquery完全不一样，**jquery**的目的是统一不同浏览器的平台api，并提供简单的封装，而**layui**是提供dom操作函数库来控制ui界面，所以layui不仅是一个组件化框架，还有现成的界面组件可以使用；

# 核心功能

博客能不能做成最关键的是，能不能解析界面，总不能每次写博客都重新做一个网页吧，我决定采用解析markdown的解析，并自己做了个简易的解析器，实现了md的基本语法，代码高亮采用的是prism库，其它的都是我用正则表达式写的，目前偶尔有解析失败的情况，有些字符串串组合我可能没考虑进去，以后再慢慢改吧；
